# Scala REST API example

## Running Locally

Make sure you have Play and sbt installed.  

```sh
$ git clone https://github.com/eltonsantos/scala-rest-api-example.git
$ cd scala-rest-api-example
$ sbt compile stage
$ sbt run
```
